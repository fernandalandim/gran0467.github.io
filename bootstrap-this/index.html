  1. Responsiveness
  2. Speed of Development

1 - Responsiveness

Creating responsive websites is so much easier with Bootstrap. The fluid grid layout that dynamically adjusts to the proper screen resolution.

2 - Speed of Development

Rather than coding from scratch, Bootstrap enables you to utilize ready made blocks of code to help you get started. So if you don't have a huge amount of knowledge in CSS you can learn how to use Bootstrap and do the website faster.
